Question 1:What is the difference among Map, Filter, ForEach and Find?

Answer
Map return a new array.
Filter return a new array of an array's element which fulfill condition.
ForEach return each value of an array.
Find return first element of an array's element which fulfill condition.

Question 2: What is the difference among Local storage, Session storage and Cookies?

Answer
Local sorage data is not cleared if the the page is open in new tab or the browser is closed or reopened.
Session storage is cleared with session ends. if the page is opened in new tab or page is closed then storage is cleared.
When a user visit a site then some information store in the browser this is cookies. It is cleared when the browser is closed.
